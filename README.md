

### 📰 News Web Application (PHP Practice Project)
**News platform where users can post stories, comment on articles, and interact with content through a voting system.**

> This project was created as a practice project to improve my experience with PHP backend and web applications. I built a news platform where users can register, log in, submit stories with titles, descriptions, and links, comment on posts, and interact with content through a voting system. Stories and comments include timestamps and user information so that readers can follow when content was posted and who contributed it. Users can browse submitted stories, read discussions in the comment sections, vote on content and participate in conversations under each article. Registered users can also edit or delete their own stories and comments. The application handles user authentication, database interactions, and dynamic content rendering using PHP and MySQL.

> **Independent Work**  
> **My Role:** Implemented user authentication, story and comment management features, voting functionality, and database interactions using PHP and MySQL.

> **Tech Stack**  
> - **Frontend:** HTML, CSS  
> - **Backend:** PHP  
> - **Database:** MySQL
